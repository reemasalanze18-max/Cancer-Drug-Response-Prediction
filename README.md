# Cancer Drug Response Prediction Using Machine Learning

## Overview

This project investigates the use of machine learning models to predict cancer drug response using genomic data from the Genomics of Drug Sensitivity in Cancer (GDSC) database.

## Objectives

* Predict drug sensitivity using gene expression data.
* Compare the performance of multiple machine learning models.
* Identify genomic features associated with drug response.
* Support precision medicine through computational analysis.

## Dataset

* 10,297 cancer samples
* 10,000+ genomic features
* Drug response measurements (LN_IC50)

## Methods

* Data Cleaning and Preprocessing
* Feature Engineering
* Exploratory Data Analysis (EDA)
* Machine Learning Model Development
* Model Evaluation

## Models Evaluated

1. Linear Regression
2. Ridge Regression
3. Random Forest
4. XGBoost

## Results

* Best Model: Ridge Regression
* R² Score: 0.82
